A from-scratch terminal, drawn cell-by-cell, in a single binary.

⇄

Multi-Agent Relay

/crew opens a pane where claude, codex, and opencode message each other to work a task — agents discovered at runtime, with loop guards, per-call timeouts, and every hop logged.

▦

Auto-Tiling Panes

Panes auto-tile into a near-square grid. Each has a title bar with its index, program title, and status glyphs. No overlays — everything is a tile.

☀

GPU Cell Rendering

Every cell is drawn on the GPU with winit, wgpu, and glyphon. Panes have SDF rounded borders; the canvas is pure black with natural ANSI colors.

⌨

Per-Pane PTY

Each pane runs a real terminal backed by alacritty_terminal and portable-pty. Shells launch as your $SHELL login shell, so your full config loads.

▸

Docked Input Bar

A command bar with slash commands, fish-style autosuggest, cd completion with $VAR expansion, and persisted history.

☰

Live Sidebar

A docked panel with a clock, CPU/MEM/DISK gauges, load average, host info, network rates, a git section, and a clickable list of open panes.

🔎

Scrollback & Find

Mouse wheel or Shift+PageUp/Down scroll a pane's history. /find <text> jumps back to the most recent matching line with smart case.

⚙

Live Settings

/settings opens a form for font family, font size, and the sidebar. Changes persist to a TOML config and apply live on Save.

🌐

Cross-Platform

One native binary for macOS, Linux, and Windows. Clipboard via Cmd+C/V and OSC 52, plus chat and agent plugins.
