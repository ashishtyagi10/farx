Everything you need. Nothing you don't.

Power-user features in a single, dependency-free binary.

☰

Dual-Panel Navigation

Side-by-side panels with tree view, file previews, and one-key copy/move between directories.

✎

Built-in Editor

Syntax highlighting, undo/redo, search & replace. Edit files without leaving the manager.

🔎

Powerful Search

Find files by name pattern or content across entire directory trees. Jump to results instantly.

⚙

Fully Configurable

TOML-based config for keymaps, themes, sorting, hidden files, external editors, and more.

🚀

Blazing Fast

Written in Rust with zero garbage collection. Instant startup, smooth 60fps rendering.

🌐

Cross-Platform

One binary for macOS, Linux, and Windows. Native terminal rendering via Crossterm.

🤖

AI Assistant

Ask questions in plain English. Supports OpenRouter, Anthropic, OpenAI, and local Ollama.

↻

Auto-Updates

Install once, stay current forever. Background update checks with one-command upgrades.

Up and running in 10 seconds.

macOS & Linux

curl -sSfL https://raw.githubusercontent.com/ashishtyagi10/crew/main/install.sh | sh

Build from Source

git clone https://github.com/ashishtyagi10/crew.git cd crew && cargo build --release

Update

farx --update
